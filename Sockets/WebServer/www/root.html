You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply-- returns product of 2 integers given as parameters e.g. multiply?num1=5&num2=7
  • /github-- returns repo data. Full_name, ID, and login of the owner. e.g. github?query = users/amehlhase316/repos
  • /calculate mortgage-- returns monthly payment for given parameters. e.g. http://localhost:9000/calculateMortgage?loanAmount=200000&interestRate=3.5&loanTerm=30
  • unitConversion-- returns value in given units. options are, km-miles, kg-lbs, and celsius-farenheit. It also does the reverse. e.g. http://localhost:9000/convertUnit?value=100&fromUnit=km&toUnit=miles
